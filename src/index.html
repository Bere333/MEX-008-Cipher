¿Tienes un descubrimiento?

Ayuda sobre funcionamiento de Boson

Boson es una aplicación web que encripta y desencripta descubrimientos científicos. Es necesario proporcionar un offset para ambas opciones

Encripta tu descubrimiento y compartelo con tus colegas

Desencripta un descubrimiento compartido

Instrucciones para encriptar descubrimiento

  1. Escribir descubrimiento a encriptar
  2. Proporcionar un número de desplazamientos (Offset)
  3. Una vez introducido el mensaje, presionar el boton encriptar
  4. Copiar mensaje encriptado
  • Nota: El mensaje no debe incluir espacios, letras minúsculas, signos de puntuacion y demas caracteres

Instrucciones para encriptar descubrimiento

  1. Escribir descubrimiento a desencriptar
  2. Proporcionar un número de desplazamientos (Offset)
  3. Una vez introducido el mensaje, presionar el boton desencriptar
  4. Copiar mensaje desencriptado
  • Nota: El mensaje no debe incluir espacios, letras minúsculas, signos de puntuacion y demas caracteres
